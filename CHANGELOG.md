# Changelog

## [1.0.0] - 2026-04-29

### Added
- **기획/아키텍처 문서화:** `docs/PRD.md`, `docs/ARCH.md` 작성.
- **의존성 업데이트:** `streamlit`, `yfinance`, `beautifulsoup4`, `FinanceDataReader` 등 대시보드 구동에 필요한 패키지 추가.
- **데이터 수집 모듈 (`src/data_fetcher.py`):**
  - 미국 시장 핵심 지수 (S&P 500, NASDAQ, Dow, 등) 실시간 요약 데이터 조회 연동.
  - 하루치 시가/종가 흐름을 보여주는 미니 차트용 분봉 데이터(intraday) 연동.
- **주도 테마 분석 모듈 (`src/theme_analyzer.py`):**
  - 네이버 금융 테마 시세 웹 스크래핑 로직 구현.
  - 당일 등락률 기준 상위 10개 주도 테마 선정 및 주도주 정보 추출 기능 추가.
- **UI 컴포넌트 (`src/components.py`):**
  - 지수 요약(`st.metric`) 및 라인 차트(`st.line_chart`)를 조합한 미니차트 UI 컴포넌트 작성.
- **메인 애플리케이션 (`src/main.py`):**
  - 전체 레이아웃 구성 및 대시보드 뷰 통합.

### Changed
- **주도 테마 상세 데이터 분석 로직 변경 (`src/theme_analyzer.py`):**
  - 기존 테마별 단순 등락률 및 주도주 텍스트 출력 방식에서, 각 테마의 상세 페이지를 스크래핑하여 개별 종목별(종목명, 현재가, 등락률, 거래대금) 세부 데이터를 수집하도록 고도화.
  - 너무 많은 데이터가 노출되는 것을 방지하기 위해 상위 5개 테마, 각 테마별 최대 10개 종목으로 수집 범위 축소.
  - 거래대금 1,000억 원(100,000 백만) 이상 여부를 판별하는 `is_high_volume` 플래그 추가.
- **주도 테마 UI 변경 (`frontend/app.js`, `index.html`, `style.css`):**
  - 테이블 레이아웃을 `테마명 | 종목명 | 현재가 | 등락률 | 거래대금` 컬럼 형식으로 재구성하고, '거래대금(백만)'으로 단위 명시.
  - 단일 테이블 구조에서 각 테마별 독립적인 표(Table)를 렌더링하는 방식으로 변경.
  - 테마명 셀 텍스트를 가운데 정렬하고 너비를 축소.
  - 거래대금이 1,000억 이상인 종목 행에 하이라이트(`.high-volume-row`) 처리 추가.
- **전체 UI 라이트 모드 전환 (`frontend/style.css`):**
  - 어두운 배경색을 흰색 계열(`#ffffff`, `#f9fafb`)로 전면 교체.
  - 테이블 테마명 백그라운드 컬러를 제거(투명 처리)하고, 텍스트 컬러를 라이트 테마에 맞게 조정.
- **시장 지표 정확도 및 차트 UI 개선 (`src/data_fetcher.py`, `frontend/app.js`):**
  - KOSPI 및 KOSDAQ 지수 요약 데이터에 대해 `yfinance` 대신 한국 시장 특화 패키지인 `FinanceDataReader`를 사용하여 오늘 종가 기준 정확도 향상.
  - 미니 차트 상단에 `시작일`과 `종가`를 텍스트로 추가 표시하여 기준점을 명확화.
  - 차트 툴팁 배경을 라이트 모드 컬러로 변경.
