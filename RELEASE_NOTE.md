# 주식 대시보드 릴리즈 노트 (v1.0.0)

**배포 일자**: 2026-04-29

## 🚀 새로운 기능 (New Features)
- **주요 지표 대시보드**: 미국(S&P 500, NASDAQ, 다우 등), 국내(KOSPI, KOSDAQ), 원자재(WTI), 변동성 지표(VIX)의 현재가, 등락률, 그리고 일일 시세 흐름을 보여주는 **미니 라인 차트**를 구현했습니다.
- **주도 테마 실시간 스크래핑**: 네이버 금융 테마 시세를 크롤링하여, 당일 시장을 주도하는 상위 10개 테마와 그 주도 종목(관련주) 정보를 제공합니다.
- **캐싱을 통한 성능 최적화**: API 호출 및 웹 스크래핑 결과에 메모리 캐싱(`cachetools`)을 적용하여 빈번한 새로고침 시에도 지연 없는 화면 렌더링을 보장합니다.

## [1.0.0] - 2026-04-30 (Production Release)

### 🚀 기능 (Features)
- **전체 시장 개편**: 미국 시장과 한국 시장 탭을 나누어 독립된 레이아웃 제공.
- **섹터 및 테마 시황**: 핀비즈(Finviz) API를 연동하여 미국 주식 시장의 섹터 및 세부 테마 흐름 시각화 보드 추가.
  - 이름, 등락률 순 정렬 기능 지원.
  - 전체 보기 및 하위 3개 테마 요약 보기 토글 지원.
- **지표 마감 시간 보정**: yfinance 연동 시 한국 시간 기준 `14:55` 멈춤 현상을 감지하여 정규장 마감인 `15:30`으로 자동 렌더링되도록 버그 픽스.

### 🛠 배포 (Deployment)
- **운영 서버 배포 환경 구축**: 
  - Ubuntu 22.04 LTS (ARM64) 환경 배포 준비 완료.
  - Gunicorn과 Nginx 리버스 프록시 조합으로 80 포트를 통한 상용 서비스 환경 세팅 완료.
  - `deploy.sh` 원클릭 배포 자동화 스크립트 제공.
  - 배포 가이드 문서 (`docs/deploy_guide.md`) 추가.

---

## 🛠 실행 및 배포 방법 (How to Run)
1. Python 환경 (3.8 이상 권장) 준비
2. `requirements.txt` 패키지 설치: `pip install -r requirements.txt`
3. 구동 스크립트 실행: `./run.sh` 또는 `streamlit run src/main.py`

## 📝 알려진 제약 사항 (Known Issues / Limitations)
- **야간선물 데이터**: 무료 API로 조회하기 어려운 야간선물 데이터는 1.0.0 버전에서 제외되었으며, 향후 전용 API가 도입될 경우 활성화될 예정입니다.
- **미니 차트 시간 지연**: 무료 금융 API(`yfinance` 등)를 이용하므로, 시장 개장 직후 초기에는 실시간 분봉 데이터 반영에 약간의 시간 지연이 있을 수 있습니다.
